# Phase quick-260508-01a Plan 01: Rename Browse / Ask page headings to Korean labels — Summary

Two visible page heading literals on `/browse` and `/ask` renamed from the English placeholders `Browse` / `Ask` to the Korean strings `Platform 브라우저` / `AI 질문하기`, matching the topbar tab labels (single source of truth in `app_v2/templates/_components/topbar.html` lines 25 and 29).

## What changed (file + line + before/after)

### Task 1: Browse page heading

**File:** `app_v2/templates/browse/index.html` line 30

**Before:**
```html
      <b>Browse</b>
```

**After:**
```html
      <b>Platform 브라우저</b>
```

The space inside `Platform 브라우저` is a regular ASCII space (U+0020), byte-for-byte identical to topbar.html line 25 (`<i class="bi bi-table"></i> Platform 브라우저`).

### Task 2: Ask page heading

**File:** `app_v2/templates/ask/index.html` line 76

**Before:**
```html
      <h1 class="panel-title">Ask</h1>
```

**After:**
```html
      <h1 class="panel-title">AI 질문하기</h1>
```

Byte-for-byte match with topbar.html line 29 (`<i class="bi bi-chat-dots"></i> AI 질문하기`); no surrounding whitespace inside the `<h1>` tag, no trailing punctuation.

## Untouched (intentional)

Per the orchestrator's `scoping_already_done` and the plan's explicit do-not-touch list, the following are preserved verbatim:

- **Internal Python identifiers** — `BrowseViewModel`, route function names (`get_browse`, `get_ask`, etc.), service-layer class names. These are code, not user-visible text.
- **`page_title` strings passed from routers** — out of scope; the page heading and the `<title>` tag have separate concerns. Browser tab title remains untouched.
- **File-purpose comments at the tops of both templates** — `app_v2/templates/browse/index.html` lines 1-23 and `app_v2/templates/ask/index.html` lines 1-27 mention "Browse" and "Ask" multiple times in architecture-describing comments and ASCII-art diagrams. Those describe the page's role, not user-visible copy.
- **`aria-label="Browse presets"` on the Browse preset row** (line 62 of `app_v2/templates/browse/index.html`) — labels the preset row affordance, not the page heading. Out of scope for this rename.
- **The Ask page ASCII-art docstring at line 10** which contains the literal `<h1 class="panel-title">Ask</h1>` inside a `{# ... #}` Jinja comment block — preserved per the plan's explicit instruction.
