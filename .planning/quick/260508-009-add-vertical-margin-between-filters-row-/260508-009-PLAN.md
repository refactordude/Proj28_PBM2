<objective>
Add visible vertical separation between the filters row and the Presets row on the Joint Validation tab so it matches the Browse (Platform 브라우저) tab.

Purpose: Browse already opens a ~16px gap between its filter row and its Presets row via Bootstrap's `mb-3` utility on the inner controls div (`<div class="d-flex align-items-center gap-2 mb-3" id="browse-filter-controls">` — `app_v2/templates/browse/_filter_bar.html:19`). Both filter-bar wrappers (`.browse-filter-bar`, `.overview-filter-bar`) explicitly zero their bottom padding in `app.css` (lines 172, 377), confirming the gap is template-level by design — Browse opts in via `mb-3`, JV currently does not.

Output: One-line class-list change in `app_v2/templates/overview/_filter_bar.html` adding `mb-3` to the form, producing a visible 1rem gap below the JV filter row above its `.ff-preset-row`.
</objective>
