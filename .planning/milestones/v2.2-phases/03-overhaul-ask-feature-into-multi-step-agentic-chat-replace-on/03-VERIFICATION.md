# Phase 3: Overhaul Ask feature into multi-step agentic chat — Verification Report

**Phase Goal:** Replace the v2.0 Phase 6 one-shot Ask page with a multi-step PydanticAI tool-using agent loop. The agent runs multiple SQL queries, inspects schema, samples distincts, reasons across tool results, and emits a structured final answer. SSE streaming via sse-starlette pushes `thought` / `tool_call` / `tool_result` / `final` / `error` events to an HTMX-driven chat surface. Cooperative cancellation via Stop button, guard-rail rejection retry (cap=5), step-budget enforcement (default=12), ephemeral 6-pair session history, NL-05 confirmation flow deleted in same atomic commit, LLM dropdown carried forward verbatim from Phase 6.
