Target public API for app/services/result_normalizer.py:

```python
from enum import Enum
import pandas as pd

class ResultType(Enum):
    MISSING = "missing"
    ERROR = "error"
    HEX = "hex"
    DECIMAL = "decimal"
    CSV = "csv"
    WHITESPACE_BLOB = "whitespace_blob"
    COMPOUND = "compound"
    IDENTIFIER = "identifier"

MISSING_SENTINELS: frozenset[str | None]           # per DATA-01
SHELL_ERROR_PREFIXES: tuple[str, ...]              # per DATA-01

def is_missing(val) -> bool: ...
def normalize(series: pd.Series) -> pd.Series: ...   # Stages 1-2
def classify(val) -> ResultType: ...                  # Stage 2 (no coercion) — DATA-02
def split_lun_item(item: str) -> tuple[int | None, str]: ...           # DATA-03
def split_dme_suffix(item: str) -> tuple[str, str | None]: ...         # DATA-04 (side detection)
def unpack_dme_compound(val: str) -> dict[str, str]: ...                # DATA-04 (value decomp)
def try_numeric(series: pd.Series) -> pd.Series: ...                    # Stage 5 — used by chart path only
```

Classification heuristics (DATA-02):
- MISSING: `is_missing(val)` is True
- ERROR: stripped string starts with any `SHELL_ERROR_PREFIXES` element
- HEX: matches `^0[xX][0-9a-fA-F]+$`
- DECIMAL: matches `^-?\d+(\.\d+)?([eE][-+]?\d+)?$`
- CSV: contains a comma AND no `=` (comma-separated list of atomic tokens)
- COMPOUND: contains `=` AND contains `,` (e.g. `local=0,peer=1`)
- WHITESPACE_BLOB: contains `\n` OR (length > 40 AND contains multiple spaces)
- IDENTIFIER: everything else (fallback)
